# Student Management App (React.js)

A lightweight, modern and interactive **Student Management Interface** built using React.js and Tailwind CSS. This project focuses on component-based architecture and dynamic data handling.

##  Features
**Create a React Application**: A fully functional frontend application built with modern standards.
**Reusable Components**: Modular structure with separate components for Form and List displays.
**Dynamic Student Records**: Displays a list of students including Name, Age, and Course details.
**Add New Students**: Interactive form to capture and inject new student data into the UI.
**Delete Functionality**: Ability to remove specific records from the list dynamically.

##  Technologies Used
**React.js**: For building the user interface and managing component lifecycle.
**JavaScript (ES6+)**: For logic and state manipulation.
* **Tailwind CSS**: For a modern, responsive, and smooth UI design.
**HTML5 & CSS3**: For semantic structure and custom styling.

##  Learning Outcomes
This project was developed to demonstrate a strong foundation in:
Understanding React component structure and hierarchy.
Effective use of **JSX** and functional components.
Managing application data using **React State** (`useState`).
Rendering dynamic data efficiently in the UI.
